一、 專案基本資訊
- 專案名稱：PinkPick Shopping Network (Demo)
- 開發目標：展示全端開發能力，針對求職面試設計。
- 更新日期：2026-03-31
- 技術棧：Laravel 11.x + React + Inertia.js + Vite
- 開發環境：Laragon (PHP 8.x, MySQL, Apache)
- 資料庫現況：MySQL 已建立，包含約 80 筆 Seeded 商品測試資料，供分頁與篩選測試。

二、 本地開發環境配置 (Laragon)
1. 核心工作：自動生成 http://pinkpick.test 網域、管理 MySQL。
2. 開發 SOP：
   - 啟動 Laragon (Start All)。
   - 終端機執行 npm run dev (啟動 Vite 熱更新)。
   - 瀏覽器訪問商城首頁。

三、 目前具備的核心功能與架構 (Current Features)
- 前端模組化：Shop.jsx 擔任容器，將 Navbar, FilterSidebar, Pagination, ProductCard 抽離。
- 元件溝通機制：熟練運用 React Callbacks 達成子組件向上傳遞資料，並觸發 API 更新。
- 認證跳轉邏輯：已覆寫 Laravel Breeze 預設路徑，登入/註冊/登出統一導回 /shop。
- API 標準化：父類別 Controller 實作 sendResponse 與 sendError，統一 JSON 回傳結構。
- 進階資料庫設計 (重要)：採用 EAV 模式 (Attributes & AttributeValues) 實作動態商品規格，具備高擴充性。
- 進階防禦機制：實作錯誤攔截，具備「物件型別檢查 (防白畫面)」與「500 錯誤遮蔽」。
